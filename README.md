# Contact Management System

A simple Python program to manage contacts in memory. Add, search, view, and delete contacts easily.

## Features
- Add a contact (name + phone number)
- Search by name or phone number
- Show all contacts
- Delete a contact

## How to Run
```bash
git clone https://github.com/Iyosiprograming/contact-management-system.git
cd contact-management-system
python3 main.py
````

## Usage

Menu options in the program:

```
1 Add
2 Search
3 Show All
4 Delete
5 Exit
```

## Files

* `contact.py` → functions to add, delete, search, and list contacts
* `main.py` → main program that runs the menu
